Course Description

This course provides an introduction to the C++ programming language, and aims at establishing a solid foundation on which students can stand on when tackling more advanced topics in the future. The primary focus of the lecture is to prepare the students for the "Computational Physics" course taught in the winter semester. Although having prior experience in any programming language will facilitate the learning process, students from all backgrounds and disciplines are welcome to attend. On top of the lectures themselves, the course will have a large practical component where the students themselves will have to use C++ to solve various problems.

The course will cover the following topics:

  • The basics of programming
  • Tools and compilers
  • Data types, pointers and references
  • C++ memory management
  • Control structures and functions
  • Exception handling
  • Classes, inheritance and polymorphism
  • Short overview of Templates and the Standard Template Library

Please use this form to sign up for the course. It is not mandatory to sign up, and there is no time limit, but if people sign up and fill inn the survey I will have more information to better optimise the course content for everyone attending.

Set up and test your coding environment

Before coming to the first lecture you should set up the coding environment you want to work in and make sure it works. There will be desktops available in the lecture room with the most common tools installed, but if you are bringing your own laptop, make sure everything works. To get some ideas you can read this article and if you have further questions you can send me an e-mail or come by my office.

To test your setup, please compile and run the following simple program:


#include<iostream>

int main()
{
  std::cout << "Hello world!" << std::endl;
}

Lookup Resources

cppreference.com

The most detailed C++ reference out there and a very good place to look for both the quirks of the language as well as function signatures, class interfaces and a list of everything available in the std libraries.

cplusplus.com

Another C++ reference which is slightly more pedagogical than cppreference, but not as easy to use on a daily basis and not as thorough.

stackoverflow.com

Probably the best place to get all your C++ questions answer. One should however first see if it has been already asked as most questions have to some extent.
